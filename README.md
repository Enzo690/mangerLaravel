## Lancement du site 


<h3>Installation</h3>

- ``composer install``
- ``php artisan migrate``
- ``php artisan serve``

<h3>Envionnement</h3>

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=food
DB_USERNAME=root
DB_PASSWORD=
```
